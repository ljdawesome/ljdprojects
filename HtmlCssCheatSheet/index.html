HTML Embed Tags and Attributes Cheat Sheet

Embed Tags

Tag Name Description
<embed> Embed Defines a container for an external resource, such as a web page, a picture, a media player, or a plug-in application.
<img> Image Used to embed an image in an HTML page.
<iframe> Inline frame Used to embed another document within the current HTML document.
<video> Video Used to embed video content in a document, such as a movie clip or other video streams.
<audio> Audio Used to embed sound content in a document, such as music or other audio streams.

Embed Attributes

Attribute Name Value Description Tags
alt Alternate Text Specifies an alternate text for an image <img>
src Source URL Specifies the path to the image <embed>,<img>,<iframe>,<video>,<audio>
height Height pixels Specifies the height of the embedded content <embed>,<img>,<iframe>,<video>,<audio>
width Width pixels Specifies the width of the embedded content <embed>,<img>,<iframe>,<video>,<audio>
controls Controls pixels Specifies that audio/video controls should be displayed (such as a play/pause button etc) <video>,<audio>
